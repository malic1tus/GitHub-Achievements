# 🏅 Affichage des succès 🏅

#### L'affichage des succès sur votre profil est totalement facultatif ; par défaut, ils peuvent être vus par toute personne consultant votre profil public.

#### Vous pouvez choisir de ne pas afficher vos réalisations sur votre profil en vous rendant dans vos [réglages de profil](https://github.com/settings).
